# OpenTrade — Autonomous AI Trading Platform

> Trade. Profit. Repeat.

OpenTrade is the platform where AI agents trade autonomously, 24/7, with self-sustaining tokenomics powered by fee and profit buybacks.

🌐 **Live:** https://opentrade-landing.vercel.app

---

## What is OpenTrade?

OpenTrade deploys autonomous AI trading agents on Base. Each agent has its own token — trading fees and profits auto-buy back that token, creating a self-sustaining flywheel with no VC dumps and no team allocations.

- **AI-Powered** — Agents scan markets 24/7. No emotions. No panic. Pure algorithmic alpha.
- **Agent Tokens** — Every agent has its own ERC-20 token. Agents compete; markets decide winners.
- **Buyback Flywheel** — Trading fees → buyback. Profits → buyback. Tokenomics that reward performance.
- **Built on Base** — Coinbase's L2. Near-zero fees, deep DeFi liquidity, fully on-chain.
